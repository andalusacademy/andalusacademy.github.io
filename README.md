# نظام إدارة مركز الاندلس للتدريب

نظام إدارة أكاديمي متكامل لـ **مركز الاندلس للتدريب** — ملف HTML واحد (Single File App) مربوط بـ Supabase كقاعدة بيانات سحابية، بدون أي Backend منفصل. يغطي إدارة الطلاب، البرامج، الحضور، الدرجات، المصروفات، الحجوزات، المهام، الإشعارات، والشهادات.

## 🚀 التشغيل / النشر

المشروع عبارة عن ملف `index.html` واحد فقط — مفيش build step ولا npm install.

### النشر على GitHub Pages
```bash
git clone https://github.com/andalusacademy/andalusacademy.github.io.git
cd andalusacademy.github.io
# انسخ index.html الجديد مكان القديم
git add index.html
git commit -m "تحديث النظام"
git push
```
GitHub Pages بيحدّث الموقع تلقائيًا خلال دقيقة أو اتنين من الـ push.

### التشغيل المحلي للتجربة
افتح `index.html` مباشرة في المتصفح، أو شغّل سيرفر بسيط:
```bash
python3 -m http.server 8000
# افتح: http://localhost:8000
```

## 🗄️ قاعدة البيانات (Supabase)

### الإعداد الأول
1. ادخل على [supabase.com](https://supabase.com) وافتح المشروع `uvwarprsghwvrfmdaaqv`.
2. روح على **SQL Editor**.
3. نفّذ ملفات الـ SQL **بالترتيب** الموجودة في فولدر `/sql`:
   - `andalus_v3_1_sql_setup.sql`
   - `andalus_v3_2_sql_setup.sql`
4. نفّذ كل بلوك لوحده، وتجاهل أي خطأ بمعنى **"already exists"** — ده طبيعي لو الجدول/العمود اتعمل قبل كده.

### الجداول الأساسية المستخدمة
| الجدول | الوظيفة |
|---|---|
| `user_profiles` | بيانات المستخدمين (مدير/موظف) وربطهم بالفروع |
| `students` | بيانات الطلاب الكاملة (شخصية + الالتحاق + صورة + رقم قومي) |
| `programs` | البرامج والكورسات وأنظمتها (مدة، مصروفات، مقدم) |
| `subjects` | المواد الدراسية ودرجات النجاح |
| `attendance` | سجل الحضور والغياب اليومي |
| `grades` | درجات الطلاب لكل مادة (دور أول/تاني) |
| `procedures` | الإجراءات الإدارية بحق الطلاب (إنذار، فصل، استدعاء...) |
| `payments` | جدول المصروفات والأقساط لكل طالب |
| `expenses` | مصروفات المركز العامة |
| `leads` | الحجوزات والاستفسارات |
| `tasks` | المهام الإدارية (بتاريخ ووقت استحقاق) |
| `schedule_sessions` | جدول المحاضرات (مرتبط بالمواد والبرامج) |
| `session_log` | سجلات الجلسات/الدخول |
| `activity_log` | سجل نشاط كل المستخدمين في النظام |
| `certificate_templates` | قوالب تصميم الشهادات المرفوعة |

### Storage Buckets
| الاسم | الاستخدام | Public |
|---|---|---|
| `student-photos` | صور الطلاب الشخصية | ✅ |
| `student-files` | ملفات ومرفقات الطلاب (PDF, صور قرارات...) | ✅ |
| `certificate-templates` | تصاميم قوالب الشهادات | ✅ |

> ⚠️ لو فشل إنشاء أي bucket تلقائيًا من السكريبت، اعمله يدويًا من **Storage** في لوحة Supabase بنفس الاسم بالظبط، وفعّل **Public**.

## ✨ المميزات الكاملة

### إدارة الطلاب
- إضافة / تعديل / حذف بيانات الطلاب الكاملة
- رفع صورة شخصية + الرقم القومي
- حذف جماعي (تحديد متعدد بـ checkbox)
- استيراد دفعة طلاب من ملف Excel (مع تحميل قالب جاهز بالأعمدة الصحيحة)
- بحث وفلترة بالاسم / رقم القيد / الفرع / البرنامج

### البرامج والكورسات
- إضافة برنامج بنظام واحد (كورس) أو عدة أنظمة (برنامج)
- تعديل أي برنامج موجود (مش بس حذف)
- أي برنامج جديد يظهر **تلقائيًا** في كل القوائم المرتبطة: تسجيل طالب، الحجوزات، فلتر المصروفات، فلتر الأوائل، جدول المحاضرات

### الحضور والغياب
- تسجيل حضور يومي بكشف لكل برنامج
- تسجيل حضور مباشر من داخل المحاضرة نفسها (بالضغط عليها في الجدول)
- رسالة واتساب تلقائية لولي الأمر عند تسجيل غياب

### الدرجات والمواد
- درجات لكل مادة (دور أول / دور تاني) مع حساب ناجح/راسب تلقائي
- امتحان تحديد مستوى إنجليزي مستقل
- عرض "الأوائل" تلقائيًا بعد كل امتحان

### المصروفات والأقساط
- جدول أقساط تلقائي حسب النظام المختار وقت تسجيل الطالب
- غرامة تأخير تلقائية بعد يوم 5 من الشهر (قابلة للاستثناء)
- تتبع حالة الدفع لكل بند (شنطة أدوات، فتح ملف، أقساط، كورس إنجليزي...)
- رسالة واتساب تلقائية بالمبلغ والتاريخ بعد كل دفعة
- تقرير مالي شامل للمركز + تصدير Excel لكل الجداول

### الإجراءات
- تسجيل إجراءات إدارية (تأخر مصاريف، استدعاء، إنذار، فصل)
- إرسال رسالة واتساب تلقائية لولي الأمر فور إضافة أي إجراء

### الحجوزات والاستفسارات (Leads)
- تسجيل حجز/استفسار جديد مع رسالة واتساب تلقائية "تم استلام طلبكم"
- زر **"تحويل لطالب"** بجانب كل حجز: يفتح فورم الطالب مع تعبئة البيانات تلقائيًا
- بعد التحويل: تحديث حالة الحجز لـ"تحول لطالب" + رسالة واتساب قبول تلقائية ("الرجاء الحضور خلال 48 ساعة")

### المهام والإشعارات
- إضافة مهام بتاريخ **ووقت** استحقاق محدد
- بوب أب تذكير تلقائي للمهام المستحقة، يفضل ظاهر لحد ما يُلغى يدويًا (مخزّن في قاعدة البيانات)
- جرس إشعارات أعلى الصفحة يجمع: مدفوعات متأخرة، نسبة غياب عالية، مهام مستحقة

### الشهادات
- بناء قالب شهادة من صورة مرفوعة
- سحب اسم الطالب على القالب وتحديد مكانه (حجم وموضع الخط)
- طباعة شهادة مباشرة لكل طالب

### النظام العام
- تسجيل دخول/خروج عبر Supabase Auth (سحابي، من أي جهاز)
- صلاحيات مدير / موظف، وربط كل موظف بفرع
- سجل نشاط كامل لكل العمليات في النظام
- حاسبة وتقويم ميلادي/هجري في الواجهة الرئيسية
- رسائل واتساب جماعية (Bulk WhatsApp)
- حماية من XSS وفحوصات أساسية على المدخلات (هاتف مصري، ID آمن)
- حماية ثلاثية الطبقات ضد تكرار البيانات (race conditions) عند الإضافة
